# Xanadu Art NGO Website

A professional, SEO-optimized website for Xanadu Art NGO - a non-profit organization based in Ohrid, Macedonia, dedicated to empowering youth through art, culture, and education.

## Features

- **Multilingual Support**: Macedonian (MK) and English (EN) versions
- **SEO Optimized**: Meta tags, Open Graph, sitemap, robots.txt
- **Accessibility**: WCAG 2.2 AA compliant
- **Responsive Design**: Mobile-first approach
- **Performance**: Optimized for Core Web Vitals

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 3.4
- Static Export

## Pages

- **Home**: Mission, focus areas, featured projects
- **About**: Organization story, values, activities, expertise
- **Projects**: Year-based project listing
- **Project Detail**: Individual project pages with gallery
- **Contact**: Contact form and information

## Color Palette

- Primary Dark: `#540863`
- Primary: `#92487A`
- Accent: `#E49BA6`
- Highlight: `#FFD3D5`
- Background: `#FFFFFF`

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
